## Conditional statements

```txt
if x > 0 {
    io.println("positive")
} else if x == 0 {
    io.println("zero")
} else {
    io.println("negative")
}
```

## Pattern matching

A7 supports `match` for branching by value/pattern.

```txt
match state {
case Idle => io.println("idle")
case Running => io.println("running")
default => io.println("unknown")
}
```

## Loops

### `while`

```txt
i := 0
while i < 10 {
    i += 1
}
```

### C-style `for`

```txt
for i := 0; i < 10; i += 1 {
    io.println(i)
}
```

### `for in`

```txt
for n in numbers {
    io.println(n)
}
```

## Jump statements

- `break`
- `continue`
- `ret`

Compiler semantic pass validates scope usage (for example, `break` outside loops is rejected).

## Control-flow validation

Semantic checks include:

- Missing returns in non-void paths
- Unreachable statement detection
- Loop control statement validity
