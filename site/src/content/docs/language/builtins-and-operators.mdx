## Builtin intrinsics

A7 builtin intrinsics use `@` prefixed identifiers.

Representative examples:

```txt
@type_set(i32, i64)
@size_of(i32)
@align_of(i64)
@type_id(i32)
@type_name(i32)
```

## Operators

### Arithmetic

- `+`, `-`, `*`, `/`, `%`

### Comparison

- `==`, `!=`, `<`, `>`, `<=`, `>=`

### Logical

- `and`, `or`, `!`

### Bitwise

- `&`, `|`, `^`, `~`, `<<`, `>>`

### Assignment

- `=`
- compound assignments like `+=`, `-=`, `*=`, `/=`

## Notes

- A7 logical operators are keywords (`and`, `or`), not C-style `&&` and `||`.
- Integer vs floating division behavior is backend-lowered according to operand types.
