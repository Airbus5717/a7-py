All examples live in `examples/`.

Run any file with this command.

```bash
uv run python main.py examples/<file>.a7
```

## Fundamentals

- `001_hello.a7` prints a greeting.
- `002_var.a7` shows mutable and immutable values.
- `003_comments.a7` shows line and block comments.
- `004_func.a7` defines and calls functions.
- `015_types.a7` uses explicit type annotations.
- `019_literals.a7` shows numeric and string literals.
- `020_operators.a7` shows arithmetic and logical operators.

## Control Flow

- `005_for_loop.a7` uses counted loops.
- `006_if.a7` uses conditional branches.
- `007_while.a7` uses while loops.
- `008_switch.a7` uses switch style branching.
- `021_control_flow.a7` combines flow statements.
- `024_defer.a7` shows deferred cleanup.

## Data And Memory

- `009_struct.a7` defines and uses structs.
- `010_enum.a7` defines enums and matching.
- `011_memory.a7` uses stack and heap values.
- `012_arrays.a7` uses arrays and indexing.
- `013_pointers.a7` shows pointer helpers.
- `016_unions.a7` uses tagged unions.
- `023_inline_structs.a7` uses inline struct literals.
- `025_linked_list.a7` builds a linked list.
- `026_binary_tree.a7` builds a tree.
- `035_matrix.a7` handles matrix values.

## Functions And Abstractions

- `014_generics.a7` shows generic functions.
- `017_methods.a7` shows method style APIs.
- `022_function_pointers.a7` passes function values.
- `027_callbacks.a7` handles callback style dispatch.

## Modules And Programs

- `018_modules.a7` imports from modules.
- `028_state_machine.a7` models explicit states.
- `029_sorting.a7` implements sorting logic.
- `030_calculator.a7` parses and evaluates expressions.
- `031_number_guessing.a7` runs a simple interactive loop.
- `032_prime_numbers.a7` computes prime values.
- `033_fibonacci.a7` computes sequences.
- `034_string_utils.a7` applies string helpers.

## Utility

- `000_empty.a7` keeps parser and pipeline smoke coverage.
