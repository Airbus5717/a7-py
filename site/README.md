# A7 Documentation Site

Astro + Starlight documentation site for the A7 compiler project.

## Local development

```bash
cd site
npm install
npm run dev
```

## Build

```bash
cd site
npm run build
```

## Check

```bash
cd site
npm run check
```

## Deployment

Deployment is handled by GitHub Actions workflow `.github/workflows/deploy-docs.yml`.

Production URL:

- `https://airbus5717.github.io/a7-py/`
